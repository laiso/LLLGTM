How To Use
===========

1. Launch LLLGTM.app
2. Drag images from browser or Finder and drop
3. LLLGTM overwrite your general pasteboard data(:exclamation:)
4. Press `⌘+V` in textarea of GitHub issues
5. :clap::clap::clap:
